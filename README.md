# ESO Adultos Málaga

Guía estática para preparar la prueba de Graduado en ESO para personas adultas en Andalucía, pensada como recopilación clara de:

- estructura oficial de la prueba
- materiales de estudio por ámbitos
- enlaces a recursos oficiales de la Junta de Andalucía
- modelos de convocatorias anteriores y recursos complementarios
- referencias locales de Málaga (IPEP Málaga y trámites)
- exámenes online con corrección por ejercicio a partir de modelos oficiales

## Stack

Sitio estático puro:

- `index.html`
- `styles.css`
- `app.js`
- `exam-data.js`
- `content/exams/` para notas y materiales estructurados del examen online
- `scripts/` para extracción/verificación desde fuentes oficiales cuando haga falta ampliar más exámenes

Sin build, sin dependencias, ideal para GitHub Pages.

## Fuentes principales

- Junta de Andalucía · Educación permanente / ESO adultos
- Junta de Andalucía · Obtención del Graduado en ESO
- Junta de Andalucía · Recursos y materiales oficiales por niveles
- IPEP Málaga · información ESPA
- Recursos complementarios de práctica claramente etiquetados como no oficiales

## Nota

La web está pensada como guía práctica. La convocatoria oficial, normativa, sedes, calendario y equivalencias siempre deben verificarse en la Junta de Andalucía antes de tomar decisiones.
